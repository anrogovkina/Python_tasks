## Фигуры на плоскости

Реализация фигур на плоскости с соблюдением принципа подстановки Барбары Лисков.
## Описание
Программа предназначена для работы с геометрическими фигурами: прямоугольник и квадрат. Нарушение принципа Барбары Лисков исправлено путем добавления свойств в класс Square. Свойства позволяют синхронизировать высоту и ширину квадрата: при изменении высоты меняется и ширина, и наоборот.

## Классы
- **Shape** — базовый класс геометрических фигур
- **Rectangle** — прямоугольник (наследуется от Shape)
- **Square** — квадрат (наследуется от Rectangle)

## Методы
- area() - площадь фигуры

## Пример использования
### Создание фигур
    rectangle = Rectangle(2, 3)
    square = Square(2)
### Вычисление площади
    square.area()
    rectangle.area()
### Изменение стороны квадрата:
    square.width = 3
