## Кэширующий декоратор
Реализация кэширующего декоратора. 

## Описание
Декоратор запоминает результаты вызова функции. При повторном вызове с теми же аргументами результат берётся из кэша, а не вычисляется заново. Декоратор может декорировать несколько функций одновременно.

## Параметры
- **depth** — максимальная глубина кэша
- **ttl** — время жизни кэша 

## Методы
- **cache_stats()** — статистика работы кэша
- **clear_cache()** — очистка кэша

## Пример использования

### Декорирование
    @cache_decorator(3, 1)
    def test_function(x):
        return x ** 2
### Статистика
    stats = test_function.cache_stats()
    for key, value in stats.items():
        print(key, value)
### Очистка кэша
    test_function.clear_cache()
